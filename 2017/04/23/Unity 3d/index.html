##游戏设计

包的使用

首先将包放到Editor里面，打开Unity,导入相应的包；

###脚本代码路径添加：（图片添加有些问题，大家参考文字叙述吧！）

  • 添加路径

    assert->scirpt->右击->create>c#文件：
    总共四个文件,main,move,weapon,bullet;(千万注意添加后不能随意改变文件名，否则不匹配无法实现效果)

  • main/move添加位置：

    首先点击assert中的screen->打开里面的test文件。
    这时候人物出现，也可以进行玩家视野的调控了。这里按说应该是可以直接将main/move拖进Player的，可是我的拖进去运行一遍自动消失，不知道别人的会不会有这个问题，然后，我用的另外一种办法,直接点击player,
    Add Component main/move,这两个c#脚本添加完成了。

  • weapon添加
    然后添加weapon到player下的（打开到底）shot-point里面；

  • bullet添加
    直接在assert出搜索installbullet,添加进去。

遇到问题

  • 我这边每次把相应的代码拖到Player，运行一遍人物属性那自动就删了，同学的就没事，后来知道用Add Component可以直接添加所要加入的c#代码，
    并且不会因为运行而出现自动清除问题。如果你遇到相同问题可以试一下第二种方法。
  • 上课时我的电脑出了些问题，所以上课时部分代码打错，后来进行了修改。
  • 我这边每次新打开文件，以前除了代码，做的其他东西，如人物运动的设置就会出现问题，就会自动消失。我问了下同学，他们每次重新打开都不会消失，于是我就都做完，直接打包给学长了。同学都是win10，我感觉不是系统问题。并且这次作业较为简单，所以我就没再去重装软件。只是重新下载并导入practice,发现还是那样。就直接做完了。
  • 人物的运动出了不少问题，但就去慢慢调节尝试就好了。90度一个方位，就好了。
    ###感悟：
    第一次接触游戏开发,很新奇，原来那些场景的开发初学者还可以直接导入应用，要不然入手难度就大多了。要注意一些细节的控制。很多时候因为一丝的疏忽导致很大的错误。得不偿失。
