    所需工具

    • pycharm
    • anaconda2 (Python2.7)
    • phantomJS

  • pycharm是python的IDE，安装即可

  • anaconda2和Python2.7区别：

    anconda2里面集成了很多关于python科学计算的第三方库，主要是安装方便且功能强大。
python是一个编译器，下载这个还需要安装python库，pip install python，添加python到环境变量中去。

    ** 注意：anaconda2和Python2.7两个，二选一安装。二选一。。

  • phantomJS 是一个无界面浏览器，无需解压，直接安装作为工具工具使用即可。
    ##安装库

如果你选择anaconda,只需要安装selenium库：命令行输入：
pip install selenium
如果你选择官方python, 所有使用到的库都要去安装,做爬虫的话，还安装BeautifulSoup:
pip install bs4

特别需要注意的地方：

  • 格式：一定要严格注意python书写格式，很多时候的bug都是由格式错误造成的。所以要先去了解格式，再去写。并且出现的格式错误很多报这个错误（我这边出现的），百度的话，说库安装问题。其实没有。

两个IT术语BUG和DEBUG：

  • BUG是指程序错误或漏洞，就是程序员编程时由于某些原因，因程序语句写错或考虑不周而未能实现想要的功能，甚至不能运行程序。最常见的可能就是某些游戏，由于程序没编好，别人就可能会利用其漏洞篡改游戏数据。
  • DEBUG就是指程序员对编好的程序进行漏洞的排查，尽量消除错误或漏洞。一般都是用专业的工具进行。
